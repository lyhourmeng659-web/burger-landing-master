# 🍔 Burger Landing Master

A modern restaurant landing page built with **Java Spring Boot**, **Thymeleaf**, **Spring Data JPA**, and **MySQL**.

The application displays burger menu items dynamically from a MySQL database and provides an attractive restaurant website interface including menus, events, happy hour promotions, contact information, and location details.

---

## 🚀 Features

- Dynamic menu management using MySQL
- Spring Boot MVC architecture
- Thymeleaf template engine
- Spring Data JPA integration
- Responsive restaurant landing page
- Category-based food display
  - Main Dishes
  - Desserts
  - Drinks
  - Happy Hour Specials
- Event showcase section
- Contact and subscription section
- Google Maps integration
- Modern UI with animations
